To create a virtual environment enter: python -m venv <env-name> ===> To activate the virtual environment enter: <env-name>\Scripts\activate  ===> To deactivate, simple enter: deactivate

***All of the following actions must have virtual environment activated***
If you install new packages, remember to list them out in the requirements.txt file by entering: pip freeze > requirements.txt

To install all packages that are listed in the requirements.txt file, simple enter: pip install -r requirements.txt
